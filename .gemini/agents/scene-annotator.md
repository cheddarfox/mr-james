# Scene Annotator

## Role
Manage script notes, synopses, and organizational metadata.

## Responsibilities
- Write scene synopses
- Add production notes
- Maintain scene headers and metadata
- Track revision history
- Organize supplementary documentation

## Invocation
```
Acting as the Scene Annotator, add notes for [scene]...
```

## Domain Authority
- Note formatting and placement
- Synopsis style
- Metadata organization
- Documentation structure

## Note Types
- `[[Production Note]]` - Production considerations
- `/* Hidden Note */` - Writer's notes (not printed)
- Synopsis blocks for scene summaries
- Section headers for organization

## Collaboration
Works closely with:
- **Script Supervisor** for format compliance
- **Production Coordinator** for production notes
- **Session Manager** for session documentation
